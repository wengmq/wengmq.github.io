## 一、简介

显卡（Graphics Card，GPU，Graphics Processing Unit 通常泛指含 GPU 的显卡）是计算机中的一个关键硬件，主要负责图形和图像的渲染计算，把 CPU 处理的数据转换成可以在显示器上显示的画面。它在游戏、视频处理、AI 深度学习、科学计算等领域都非常重要。

  ![](assets/Pasted%20image%2020250919145309.png)
## **二、显卡的组成**

1. **GPU 核心（Graphics Processing Unit）**
    
    类似 CPU 的“大脑”，负责执行图形渲染、并行计算。
    
2. **显存（VRAM, Video RAM）**
    
    储存渲染过程中需要用到的纹理、帧缓存、AI 模型参数等。
    
3. **供电与散热系统**
    
    包括供电相数、电容电感，风扇、散热片、水冷等。
    
4. **输出接口**
    
    HDMI、DisplayPort、DVI、VGA 等。
    
5. **PCB 电路板与供电接口**
    
    PCIe 接口（插在主板上），外接 6Pin/8Pin/12Pin 电源接口。


## **三、显卡的主要物理参数**

这些参数是选购和评估显卡性能的关键指标：
#### **GPU 核心相关**

- **制程工艺（nm）**：越先进功耗越低、性能越强，例如 5nm、7nm。
    
- **CUDA 核心/流处理器数（NVIDIA/AMD）**：并行计算单元数量，数量越多，理论算力越强。
    
- **核心频率（MHz/GHz）**：GPU 运行速度。
    
- **光追核心、Tensor 核心（NVIDIA 特有）**：专门加速光线追踪、AI 计算。
#### **显存（VRAM）**

- **显存容量（GB）**：常见 4GB–48GB，决定能加载多大场景或模型。
    
- **显存类型**：GDDR5、GDDR6、GDDR6X、HBM2e，速度和带宽差异大。
    
- **显存位宽（bit）**：如 128bit、192bit、384bit，越宽带宽越大。
    
- **显存带宽（GB/s）**：等于显存频率 × 位宽 ÷ 8，影响数据传输速度。
    
#### **接口与总线**

- **PCIe 通道数/版本**：PCIe 3.0 ×16、PCIe 4.0 ×16、PCIe 5.0 ×16。
    
- **视频输出接口**：HDMI 2.1、DP 1.4a、DVI、USB-C（VR）。
    
#### **功耗与散热**

- **TDP 功耗（W）**：典型 75W–600W，需要对应电源。
    
- **供电接口**：6Pin/8Pin/12Pin，决定能供给多少瓦数。
    
- **散热方式**：单风扇/双风扇/三风扇/水冷。

#### **尺寸与物理形态**

- **长度/厚度（mm，几槽位）**：决定是否能装进机箱。
    
- **重量**：部分旗舰显卡需支架防下垂。
    
#### **计算性能指标**

- **单精度浮点性能（TFLOPS）**
    
- **双精度性能（TFLOPS）**
    
- **光追性能（RT TFLOPS）**
    
- **AI 算力（Tensor TFLOPS / TOPS）**

#### 4.2 显卡对分辨率的影响：

**分辨率越高 → 像素数量越多 → 显卡负载越大**
    
- 720p（1280×720）≈ 92 万像素
	
- 1080p（1920×1080）≈ 207 万像素
	
- 2K/QHD（2560×1440）≈ 370 万像素
	
- 4K（3840×2160）≈ 829 万像素
        
→ 4K 比 1080p 多了 **4 倍的像素点**，显卡每一帧都要算更多数据。

目前热门的AIGC对显卡的要求，主要是显卡的算力和显存容量，容量决定出图的分辨率，算力决定出图的效率，一般来说中端显卡会推荐60Ti 16GB，中高端会推荐5070Ti，高端推荐5080和5090D，Ai绘图不推荐A卡。

**以游戏为例**，显卡玩游戏最直观的表现就是帧数，一般来说游戏需求选择显卡3A高画质60帧为标准、FPS游戏144帧+为标准

- 对于**NVIDIA**的显卡，目前主流的命名方式是 GTX 接一串数字和字母。
	- 其中第一位，或者前两位表示第几代产品
	- 后两位是显卡的性能定位
	- 后缀部分有 M 和 Ti，M 是为移动平台推出的产品，而 Ti 版要比普通显卡强劲
	- 举例来说，比如 GTX 1050Ti
		- GTX 品牌的称呼，还有四种性能依次降低 GTX > GTS > GT > GF
		- 1050 中，10表示的显卡代数，10 就是10代显卡，50 表示的是显卡性能定位，1060 就要比1050要强
		- 后缀字母 Ti
			- Ti 性能加强版 m 移动端显卡 mx 移动加强版 SLI 后缀表示可以通过特殊接口将显卡并联提高工作性能 Max-Q 后缀表示该显卡是 Max-Q 设计方案，调整了硬件让新显卡能够放到轻薄本中
- ![](assets/Pasted%20image%2020250919145911.png)


- **AMD**的显卡大部分是RX开头的
	- **命名：Radeon RX <系列号><后缀>**
	- **系列号（四位数字）**
	    - 第一位数字 → **GPU 架构世代**
	        - 例如：6xxx 系列 → 第六代 RDNA 架构（RDNA 2），7xxx → RDNA 3。
	    - 第二位数字 → **性能档次**（通常越大越高端
	        - 例如：RX 6700 < RX 6800 < RX 6900。
	    - 后两位数字 → **细分型号**
	        - 用于区分同一性能档次的不同显卡（比如内存大小或小幅提升）。
	- **后缀**
	    - **XT**：增强版，核心频率、流处理器数量或显存带宽更高。
	    - **XTX**：比 XT 更高的顶级版本。
	    - **M**：移动端（笔记本）显卡。
	    - **Pro**：专业显卡，如 Radeon Pro 系列。
