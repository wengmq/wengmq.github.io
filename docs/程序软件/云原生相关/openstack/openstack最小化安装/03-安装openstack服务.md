	- Placement service – [placement installation for 2025.2 (Flamingo)](https://docs.openstack.org/placement/2025.2/install/)  
	    
	- Compute service – [nova installation for 2025.2 (Flamingo)](https://docs.openstack.org/nova/2025.2/install/)  
	    
	- Networking service – [neutron installation for 2025.2 (Flamingo)](https://docs.openstack.org/neutron/2025.2/install/)
